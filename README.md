##TouchSlide 1.0.5 [触屏左右滑动效果]
这是一个为手机端触屏滑动而产生的组件，可实现基础的左右滑动模块、焦点图效果、TAB页签效果等


##Demo地址：
http://www.imf7.com/code/touch-slide/


##他的优势：
	独立存在，无需任何JS库；

	采用CSS3动画，手机端运行更为流畅；

	兼容绝大多数浏览器；

	兼容横竖屏切换；

	兼容PC调试；

	参数配置灵活多变；

	其他优势等待大家发觉。


##浏览器兼容性：
	IE10+

	Chrome

	Firefox

	Opera

	Safari
